## 约餐小程序
> 2019-05-21 

每到中午时间，同事都会因为午餐去哪里吃为难。通常的方法是，一位同事在 chat 中，发送一个想去的餐馆，看看有哪些同事愿意同往。通常，同事间会互相询问，餐馆的
菜单，口味，价格等等。可以开发一个约餐小程序，提供以下简单功能：

- 餐馆的列表
- 创建一个约餐，里面最多可以选择三个餐馆以供选择，要加上一个截止日期
- 当接到约餐的邀请后，可以在里面选择 1-2 个
- 截止时间到以后，预约将会显示最终结果。
